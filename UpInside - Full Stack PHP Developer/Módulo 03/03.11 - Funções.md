# Funções

## functions
```php
function functionName($arg1, $arg2, $arg3)
{
	$body = [$arg1, $arg2, $arg3];
	return $body;
}
```

```php
function optionArgs($arg1, $arg2 = true, $arg3 = null)
{
	$body = [$arg1, $arg2, $arg3];
	return $body;
}
```

## global access
```php
$weight = 86;
$height = 1.92;

function calcImc()
{
	global $weight;
	global $height;

	return $weight / ($height * $height);
}
```

## static arguments
Define uma variável estática para a função.
```php
function payTotal($price)
{
	static $total;
	$total += $price;
	
	return $total;
}

$pay = payTotal(200);
$pay = payTotal(150);
$pay = payTotal(500);

echo $pay;
```

## dinamic arguments
```php
function names()
{
	$names = func_get_vars();
	$count = func_num_args();

	return [$names, 'count' => $count];
}

var_dump(names('Luffy', 'Nami', 'Sanji', 'Zoro'));
```
