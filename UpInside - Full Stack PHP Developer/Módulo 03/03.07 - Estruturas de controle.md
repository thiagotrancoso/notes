# Estruturas de controle
## if, elseif, else
```php
// Este exemplo testa o conteúdo da variável
$a = true;
$b = false;
if ($a) {
	// Se "a" é verdadeiro
} elseif ($b) {
	// Se "a" é falso e "b" é verdadeiro
} else {
	// Se "a" e "b" é falso
}
```

## isset, empty, !
### isset
Verifica se uma variável existe e é diferente de **NULL**.
```php
$name = false;
if (isset($name)) {
	echo "verdadeiro";
} else {
	echo "falso";
}

// verdadeiro
```

### empty
Verifica se uma variável **não** existe ou se o conteúdo é **false**.
Empty é equivalente a: `!isset($var) || $var == false`.
```php
$name = "Thiago";
if (empty($name)) {
	echo "verdadeiro";
} else {
	echo "falso";
}

// falso
```

## switch
```php
$content = "a";
switch ($content) {
	case 'a';
		echo 'letra a';
		break;

	case 'b';
		echo 'letra b';
		break;

	default;
		echo 'Nenhuma das alternativas anteriores';
		break;
}
```
